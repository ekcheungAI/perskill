# Claims & Contradictions Ledger — {Persona Name}

> For every major claim in this persona's SKILL.md, document: (a) the evidence supporting it, (b) the contradictory evidence that challenges it, and (c) an overall credibility verdict. This ledger is the primary tool for preventing hagiography.

**Rule:** A claim without a documented contradiction is an unverified assertion. Add it to §8 of SKILL.md only after the contradiction is recorded here.

---

## How to Use This Ledger

1. Start with each mental model in SKILL.md §4 — what evidence supports it? What contradicts it?
2. Look at the adversarial sources in `04-adversarial-distillation.md`
3. Check if the person has publicly contradicted themselves
4. Record the contradiction even if you ultimately decide the claim still holds

- "{Quote or source showing the opposite behavior or stated belief}"
- "{Case where the model failed or was explicitly abandoned}"

**Verdict:** {Hold / Reject / Reframe}
- If Hold: the contradiction is surface-level, the underlying model still holds
- If Reject: the model was wrong or the persona was wrong about themselves
- If Reframe: the model needs a qualifier (e.g., "only when X", "except when Y")
