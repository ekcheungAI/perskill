## Research Depth Legend

- `[DISTILLED]` — Full distill_templates research complete. See `skills/{id}/research/` for evidence.
- *(no badge)* — Generated from `src/lib/personas.ts` only. Run `npx tsx scripts/research/persona-research.ts {id}` to begin distillation.

## How to Install a Persona
### Claude Code / Cursor
```bash
# Copy a persona folder into your skills directory
cp -r skills/elon-musk/ ~/.claude/skills/
```

### Any LLM (System Prompt)
1. Open the persona's `SYSTEM_PROMPT.md`
2. Copy the short or full prompt
3. Paste into your AI's system prompt settings

## How to Add a New Persona
```bash
# 1. Scaffold research project
npx tsx scripts/research/persona-research.ts new-persona --type=HK_ENTREPRENEUR

# 2. Run data collection (Twitter + web)
npx tsx scripts/research/pipeline.ts handle --type=HK_ENTREPRENEUR --deep-research

# 3. Fill in distillation files (triple-verify-log.md, etc.)

# 4. Run validation
npx tsx scripts/research/run-validation.ts new-persona

# 5. Add persona to src/lib/personas.ts

# 6. Export skills
npx tsx scripts/export-personas.ts
```

## Auto-Export
This folder is auto-generated by `scripts/export-personas.ts`.  
To regenerate after updating `src/lib/personas.ts`:
```bash
npx tsx scripts/export-personas.ts
```
