## Thinking Frameworks

- **Top-Down Macro Conviction + Bottom-Up Confirmation**: Start with macro thesis (interest rates rising, currency weakening). Then confirm with bottom-up evidence (equity valuations, earnings forecasts). Only trade the intersection.
- **Risk-First Sizing**: Before asking 'how much can I make if right?', ask 'how much can I lose if wrong?' Size bets such that even maximum loss is acceptable.
